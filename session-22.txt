====================================

  task:

** Instal nginx in server Through playbook**
===============================================
- name: install and run nginx
    hosts: web
    become: yes (giving the sudo access to install )
    tasks:
    - name: install nginx
        ansible.builtin.package:(module convert into DNF At the time of Gathering facts)
            name: nginx
            state: present
